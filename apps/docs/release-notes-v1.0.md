# Release Notes v1.0 - iNest Phone

## Visao Geral

A versao 1.0 consolida o MVP da plataforma iNest Phone, com base tecnica em Next.js, NestJS, PostgreSQL e Prisma ORM.

## Entregas Principais

- Documentacao consolidada: PRD, BRD, SAD, UXS e DMS.
- Banco de dados oficial com Prisma.
- Autenticacao e controle de acesso.
- Layout base e Design System.
- Configuracoes do Sistema.
- Produtos.
- Fornecedores.
- Radar de Precos.
- Precificacao.
- Radar de Importacao.
- Gerador de Ofertas.
- Dashboard Gerencial.
- Integracoes.
- QA tecnico.
- Preparacao de deploy e Go Live.

## Limitacoes Conhecidas

- Integracoes externas reais ainda dependem de credenciais e APIs oficiais.
- Cobertura formal de testes ainda depende da configuracao de coverage do Vitest.
- E2E completo ainda depende de ferramenta dedicada.
- Banco de producao ainda precisa ser provisionado e validado.

## Decisoes Tecnicas

- Arquitetura modular em monolito.
- Backend NestJS.
- Frontend Next.js.
- Prisma ORM.
- PostgreSQL gerenciado para producao.
- Deploy previsto em Vercel e Render ou Railway.

## Status

Versao preparada para homologacao e checklist final de producao.
